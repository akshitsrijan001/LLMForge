# LLMForge

Automated DSPy-powered RAG Pipeline Optimizer.

LLMForge automatically discovers the best combination of:

- LLM
- Prompt Strategy
- Retrieval Method

for any document corpus by benchmarking multiple pipelines and selecting the best-performing configuration.

## Tech Stack

- Next.js
- FastAPI
- DSPy
- LangGraph
- LlamaIndex
- Ollama
- Redis
- PostgreSQL
- Prisma
- Supabase
