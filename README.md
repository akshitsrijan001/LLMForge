# 🚀 LLMForge

A modern multi-model AI workspace powered by Ollama and FastAPI.

LLMForge provides an intuitive ChatGPT-like interface with intelligent model routing, local LLM support, and an extensible architecture for developers and researchers.

---

## ✨ Features

- 🤖 Multi-model AI support
- ⚡ Intelligent model routing
- 🧠 Auto model selection
- 💻 Dedicated coding model
- 🚀 Powerful general-purpose model
- 📝 Markdown rendering
- 🎨 Syntax highlighting
- 📋 One-click copy responses
- 🧵 Conversation history
- 🎛️ Interactive control center
- 🌙 Modern dark UI

---

## Supported Models

| Mode | Model |
|------|-------|
| ⚡ Fast | Gemma 3 1B |
| 💻 Coder | Qwen2.5 Coder 3B |
| 🧠 Reasoning | Phi-3 Mini |
| 🚀 Powerful | Llama 3.1 8B |
| 🤖 Auto | Intelligent Router |
